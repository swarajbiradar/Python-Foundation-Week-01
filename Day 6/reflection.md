# Week 1 Python Learning Reflection

## Topics Covered
- **Day 1:** Python setup, variables, and data types (int, float, string, boolean, list)  
- **Day 2:** Operators (arithmetic, comparison, logical) and small math problems  
- **Day 3:** Control flow (if-else statements, for and while loops)  
- **Day 4:** Functions (defining functions, parameters, return values, local vs global scope)  
- **Day 5:** Mini projects (simple calculator and number guessing game)  

---

## Easy Topics
- Installing Python and setting up the environment (Jupyter Notebook / VS Code)  
- Basic data types and printing outputs  
- Arithmetic operators (+, -, *, /, %, **)  
- Writing simple loops (for printing numbers, sum calculations)  

---

## Challenging Topics
- Understanding variable scope (local vs global) in functions  
- Designing reusable functions for problems like factorial or prime checks  
- Applying loops and conditionals together in mini-projects  
- Ensuring proper input handling and program flow in projects like the number guessing game  

---

## Key Learnings
- Python syntax is straightforward, but proper indentation is crucial.  
- Loops and conditional statements are powerful tools for repetitive tasks and decision-making.  
- Functions improve code reusability and readability.  
- Practice projects (calculator, guessing game) help in understanding program structure and logic.  
- Debugging and testing are essential to ensure programs work as intended.
