
# Python Foundations - Week 1

This repository contains all exercises, examples, and mini-projects completed during **Week 1** of Python learning. The projects cover the fundamentals of Python programming, including variables, data types, operators, control flow, loops, functions, and beginner-level projects.

---

## Topics Covered

### Day 1: Python Setup & Basics
- Installed Python and development environment (Jupyter Notebook / VS Code)  
- Learned variables and basic data types: `int`, `float`, `string`, `boolean`, `list`  
- Practice Programs:
  - Examples of each data type
  - Combined program demonstrating all data types

### Day 2: Operators
- Explored **arithmetic operators**: `+`, `-`, `*`, `/`, `%`, `**`  
- Explored **comparison operators**: `==`, `!=`, `<`, `>`, `<=`, `>=`  
- Explored **logical operators**: `and`, `or`, `not`  
- Solved small math problems:
  - Area of a circle
  - Simple interest calculation

### Day 3: Control Flow
- Learned **if-else statements** for decision making  
- Practiced **loops**:
  - `for` loop
  - `while` loop  
- Practice Programs:
  - Print numbers 1–10
  - Print even numbers up to 50
  - Sum of first 100 natural numbers

### Day 4: Functions
- Learned to define functions with **parameters and return values**  
- Created **reusable functions**:
  - Factorial calculation
  - Prime number check
  - Even/Odd checker  
- Explored **variable scope**:
  - Local vs Global variables

### Day 5: Mini Projects
- **Simple Calculator** (addition, subtraction, multiplication, division)  
- **Number Guessing Game** (random number between 1–100, user guesses until correct)

### Day 6: Review & Practice
- Revised all topics from Days 1–5  
- Solved beginner-level problems:
  - Sum of array
  - Reverse a string
  - Count even/odd numbers
  - Palindrome check
